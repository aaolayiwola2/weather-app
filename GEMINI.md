# Gemini Project Instructions

This file provides instructions for the Gemini AI on how to interact with this project.

## Project Overview

This is a weather application. The main files are in the `src` directory.

## Development

- To run the application, open `src/index.html` in a browser.
- The main logic is in `src/scripts.js`.
- The styles are in `src/input.css` and compiled to `src/output.css`.

## Goals

- The user is currently working on the `switchUnit` function in `src/scripts.js`.
- Please assist the user in completing this function.
